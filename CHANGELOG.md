# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-08

### Added
- Line-level temperature marking (Comfort / Learning / Panic zones)
- Persistent SQLite storage via sql.js (marks survive vault restarts)
- Learning history sidebar panel with date-grouped entries
- Calendar heatmap with two display modes: Count and Status
- Status-change arrow indicator in history entries
- Vocabulary collection queue (Ctrl+Q to collect selected text)
- Vocabulary term highlighting with tooltip preview
- Two-way link support between vocabulary entries and notes
- CM6 StateField-based line background decorations (stable across editor rebuilds)
